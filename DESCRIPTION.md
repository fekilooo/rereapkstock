# GitHub Repo Description 建議文案

## 中文版

樂活五線譜是一款以台股、美股追蹤為主的 Android 股票 App，整合五線譜技術分析、樂活通道、Fear & Greed 市場情緒觀察與多週期圖表。支援 3M、6M、1.5Y、3.5Y 區間分析，適合想在手機上快速掌握趨勢與區間位置的投資人。

## English Version

Lohas Stock App is an Android stock tracking app for Taiwan and US markets, featuring five-line technical analysis, Lohas channel overlays, Fear & Greed sentiment tracking, and multi-period charts. It is designed for mobile users who want a fast way to review trend zones and market sentiment.
